# Installation

## Prerequisites
- Composer
- Node.js
- Homebrew (on MacOS)

## Windows
- Download WLS2
- Download Docker Desktop
- Refer to https://laravel.com/docs/9.x/sail
- Run "composer install" and "npm install"
- Run "php artisan key:generate"

## MacOS
- Download Laravel Valet via Composer
- Install MySQL/mariadb via homebrew
- Run "composer install" and "npm install"
- Run "php artisan key:generate"
